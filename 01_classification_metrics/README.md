# 模块 01：底层语义引擎 (Classification & Metrics)

## 📌 模块简介
本模块是 PetAsset-Hub 的基础识别中枢。通过在 `Oxford-IIIT Pet` 数据集上训练的深度残差网络（ResNet18），实现对输入图像或视频帧的自动化元数据打标。本模块的输出结果将指导后续管线进行本地资产的结构化目录归档（如自动建立 `Dog/Corgi` 文件夹）。

## 🚀 核心任务
本模块包含两个并行的图像分类子任务：
* `binary`：猫狗二分类（标签：`Cat=0`, `Dog=1`），用于大类初步过滤。
* `breed`：37 个细分宠物品种分类，用于生成精细化的结构资产标签。
