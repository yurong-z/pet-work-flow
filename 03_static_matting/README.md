# 模块 03：发丝级静态资产提取 (High-Fidelity Static Matting)

## 📌 模块简介
本模块是 PetAsset-Hub 的核心静态视觉资产生成器。在获取到前置模块筛选出的高清图像后，本模块负责剥离复杂的真实世界背景。为了解决传统分割算法在宠物毛发、猫咪胡须等高频边缘处的“锯齿”与“溢色”痛点，本模块引入了工业级的高精度抠图模型（如 RMBG-1.4 或 BiRefNet），实现真正意义上的发丝级零样本抠像。

## 🚀 核心任务
* **单帧高精分割**：加载预训练的高清抠图网络权重，对输入的单张 `RGB` 宠物图像进行正向推理。
* **Alpha 通道重建**：模型不仅输出二值掩码（Mask），更输出极其平滑的 `Alpha` 透明通道矩阵。
* **资产无损导出**：将提取出的宠物前景与 Alpha 通道进行融合，导出为无损、背景透明的 `.png` 贴纸资产，支持直接导入 PPT 或剪辑软件。
