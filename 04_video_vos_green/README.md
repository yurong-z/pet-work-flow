# 模块 04：时序追踪与动态绿幕 (Video Object Segmentation)

## 📌 模块简介
本模块是 PetAsset-Hub 的高阶动态视觉资产生成器。静态抠图解决了“一瞬”的资产提取，而本模块将空间域的抠图扩展至时间域（Temporal Domain）。通过视频目标分割（VOS）或连续帧掩码重构技术，系统能够追踪视频中运动的宠物主体，并将复杂背景替换为纯色，从而产出符合影视后期标准（如 Premiere / 剪映）的动态绿幕素材。

## 🚀 核心任务
* **时序序列解构**：提取 2-3 秒的短视频片段，将其分解为连续的图像帧序列。
* **时序目标分割**：针对每一帧，调用轻量级分割模型（或循环调用单帧抠图模型）提取具有时序连贯性的宠物前景掩码。
* **背景重构与合成**：将所有帧的背景替换为纯绿色 `RGB(0, 255, 0)`，并利用音视频处理库将独立帧重新合成为带有连贯动作的动态视频或 GIF 动图。
