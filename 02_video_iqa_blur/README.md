# 模块 02：视频抽帧与图像质检 (Video Parsing & IQA)

## 📌 模块简介
本模块是 PetAsset-Hub 的数据前置清洗中心。由于原生（Raw）视频在拍摄时经常存在手抖、快速移动导致的运动模糊，本模块利用 OpenCV 库对输入视频进行时序降采样（抽帧），并引入**无参考图像质量评估 (IQA)** 算法，实现废片的自动化拦截。

## 🚀 核心任务
* **视频流解构**：利用 `cv2.VideoCapture` 建立视频读取管道，按固定帧率（步长）提取静态图像。
* **清晰度量化**：对每一帧图像计算**拉普拉斯方差（Laplacian Variance）**。该指标利用拉普拉斯算子对图像进行二阶导数计算，方差越大代表图像边缘越尖锐、越清晰。
* **自动化前置过滤**：设定清晰度阈值，低于阈值的帧自动判定为“模糊废片”，高于阈值的帧判定为“高清候选帧”，分流保存。
