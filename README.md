# PetAsset-Hub 🐾

面向短视频创作者的宠物数字资产智能提取与分析工具箱 (Pet Digital Asset Extraction Toolkit)。

## 项目简介

在短视频生态中，“萌宠”赛道拥有极高的播放体量，但原生（Raw）宠物视频往往存在大量废片，且传统剪辑软件难以实现资产的垂直化提取与结构化沉淀。

**PetAsset-Hub** 是一个专为宠物短视频创作者设计的 CV 算法实验项目。本项目采用**高度解耦的模块化设计**，摒弃了繁重的端到端黑盒管线，将复杂的视觉处理任务拆解为 5 个独立的高性能算法模块。通过极低代码量的 API 调用，实现从一段原生视频中精准“榨取”多维度的商业数字资产（如分类标签、高清透明贴纸、动态绿幕、自动表情包等）。

## 环境依赖与配置

本项目统一使用 `uv` 进行现代化的 Python 环境与依赖管理，确保小组成员间环境的绝对一致性。

### 1. 基础环境安装
请确保本地已安装 Python 3.10+。在项目根目录下，运行以下命令同步虚拟环境：

    uv venv --python 3.10
    uv sync --extra notebook

### 2. 新增依赖库规范
各模块负责人若需引入第三方库（如 `opencv-python`, `pillow` 等），**严禁使用全局 pip**，请务必在根目录下通过 `uv` 添加，以同步更新 `pyproject.toml`：

    uv add <package_name>

## 核心算法模块与目录架构

项目按功能解耦为 5 个完全独立的算法验证模块，各负责人仅需维护所属目录下的代码与测试。

### 01_classification_metrics (底层语义引擎)
* **核心功能**：基于 Oxford-IIIT Pet 数据集与 ResNet18 构建的宠物分类器。
* **业务价值**：为无序的本地视频素材自动生成结构化的品种标签（如 `Breed=Corgi`），实现资产的自动化归档建库。
* **主要产出**：二分类与 37 品种分类的 Accuracy/Loss 训练曲线、混淆矩阵热力图。

### 02_video_iqa_blur (视频抽帧与图像质检)
* **核心功能**：利用 OpenCV 与 Laplacian 算子进行的视频自动化抽帧与无参考图像质量评估 (IQA)。
* **业务价值**：自动剔除手抖、运动模糊的废片，为下游分割模型提供高信噪比的清晰候选帧图库。
* **主要产出**：去糊前后的对比样张集、清晰度得分阈值分布图。

### 03_static_matting (发丝级静态资产提取)
* **核心功能**：基于前沿开源高精抠图模型（如 RMBG-1.4 或 BiRefNet）的单帧图像分割。
* **业务价值**：克服宠物毛发边缘复杂的难点，一键提取带有 Alpha 透明通道的高清商业设计贴纸。
* **主要产出**：高对比度的抠图前后效果展示、局部毛发细节放大图。

### 04_video_vos_green (时序追踪与动态绿幕)
* **核心功能**：利用多帧掩码分割技术实现连续序列帧的背景重构。
* **业务价值**：将空间域静态抠图扩展至时间域，生成影视后期标准的动态绿幕素材。
* **主要产出**：3-5秒的高质量宠物动态绿幕视频短片或动图 (`.mp4` / `.gif`)。

### 05_meme_synthesis (商业资产后处理引擎)
* **核心功能**：利用 Python 形态学操作（膨胀算法）与图像绘制模块实现自动化贴纸合成。
* **业务价值**：为透明底图自动化注入贴纸白边、底部阴影与多模态文案，打通素材到社交流量变现的最后一公里。
* **主要产出**：工业化批量生成的趣味萌宠表情包矩阵。

## 协作规范 (Git Workflow)

为避免环境冲突与大文件拉取失败，团队成员需严格遵守以下协作纪律：

1. **绝对隔离开发**：所有人均在 `main` 分支操作，但**仅限在自己负责的 `0x_...` 文件夹内修改代码**。
2. **大文件防爆雷**：仓库已配置强力的 `.gitignore`。**严禁**绕过规则强制上传任何 `.mp4`, `.pt/pth`, `.jpg/png` 等大体积媒体文件及模型权重文件。测试样例请保留在个人本地电脑。
3. **每日同步代码**：编写代码前必须先执行 `git pull origin main` 同步最新进度。
